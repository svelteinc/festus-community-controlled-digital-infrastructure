# Community Engagement Documentation

This directory contains public-facing and community-feedback materials for the Festus CCDI project.

Typical contents:
- Public meeting slide decks and talking points.
- FAQs and myth-vs-fact sheets.
- Summaries of feedback from residents and stakeholders.
- Communication plans for major project milestones.
- Plain-language explainers for social media (e.g., `festus_facebook_explainer_v1.md`).
- Simple numbers one-pagers and question lists for PNZ/Council
  (e.g., `data_center_numbers_onepager_v1.md`,
  `questions_for_pnz_and_council_v1.md`).

- Options one-pagers that compare different paths for Festus (e.g., `festus_dc_options_onepager_v1.md`).


## Where residents and civic volunteers can start

- `faq_v1.md` – main community FAQ.
- `festus_facebook_explainer_v1.md` – plain-language explainer you can post or adapt for social media.
- `festus_dc_options_onepager_v1.md` – short comparison of different paths Festus can take.
- `data_center_numbers_onepager_v1.md` – simple numbers explainer for typical data center deals vs this model.
- `festus_desoto_corridor_growth_with_guardrails_onepager_v1.md` – one-page plain-language summary of the corridor "growth with guardrails" idea.
- `festus_desoto_corridor_strategy_v1.md` – deeper context on the wider Festus + De Soto + Hematite corridor and growth with guardrails.
- `bitcoin_resident_summary_v1.md` – short explainer on Bitcoin and bitcoin mining for residents.
